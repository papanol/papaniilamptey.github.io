Scenario
Mr. John Hughes is currently searching for the model to accurately predict the extent of a students improvement, in standardized scores following training.

Goal
The goal of this project is to test and analyze 3 different models that is Naive Bayes, Neural Networks and Logistic Regression in the quest to find the model to predict a students change in standardized score after training

Data Overview

The dataset contains information on 60 observations, each representing a different case in a study.The dataset studentperf has 6 columns and 61 rows.
Dependent variable: Outcome
Independent variables: Method, satis, time, ability, prvperf

Key Statistics

Correlation Matrix

There is a weak positive correlation between the independent variables, with the strongest being 0.53 for ability and method.
There is negligible negative correlation between the independent variables with -0.36 for method and prvperf

Pairplot Chart

The scatter plots do not follow any particular diagonal trend which indicates that there is little or no correlation
There are little or few outliers which are data points that lie far away from the main cluster of points in the scatter plots.

Classification Matrix insights
Neural Network
For class 0, the F1-score is 0.63, and for class 1, it is 0.25. A higher F1-score indicates a better balance between precision and recall. Accuracy: The accuracy of the neural network on the test data is 0.50, meaning it correctly predicted 50% of the instances.
Naive Bayes
For class 0, the F1-score is 0.67, and for class 1, it is 0.44. A higher F1-score indicates a better balance between precision and recall. Accuracy: The accuracy of the Naive Bayes classifier on the test data is 0.58, meaning it correctly predicted 58% of the instances.
Logistic Regression
For class 0, the F1-score is 0.57, and for class 1, it is 0.40. A higher F1-score indicates a better balance between precision and recall. Accuracy: The accuracy of the Logistic regression classifier on the test data is 0.50, meaning it correctly predicted 50% of the instances.

Recommendation
The Model I recommend for the dataset is the Naive Bayes model. This model has the highest f1 score of 0.67 amongst the lot.The F1-score is the harmonic mean of precision and recall. It provides a balance between precision and recall, making it a useful metric when there is an uneven class distribution or when both false positives and false negatives are equally important. Also the Naive Bayes has the highest accuracy of 0.58 which represents the proportion of correctly classified instances (both true positives and true negatives) out of the total number of instances.
